# blender-static-model
A basic static model built in blender. Assignment for CS 438

# Model
This is a very basic static model using primitive objects in blender. The model is meant to represent a tower of hanoi toy that I had as a kid.

# Objects

The objects used for the model are the following:

- A cube for the base
- 3 cylinders for the poles
- 8 torus's for the rings
